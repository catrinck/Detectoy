Detectoy API

Bem-vindo à API do Detectoy. Endpoints disponíveis:

Documentação

Acesse a documentação interativa da API em: /docs

Detecção de Objetos

  • POST /api/v1/detections/ - Upload e análise de imagem
  • WS /api/v1/camera/stream - Stream de câmera em tempo real

Gerenciamento de Gerentes

  • GET /api/v1/gerentes/ - Listar todos os gerentes
  • POST /api/v1/gerentes/ - Criar novo gerente
  • GET /api/v1/gerentes/{cpf} - Obter gerente específico
  • PUT /api/v1/gerentes/{cpf} - Atualizar gerente
  • DELETE /api/v1/gerentes/{cpf} - Remover gerente

Gerenciamento de Usuários

  • GET /api/funcionarios/ - Listar todos os usuários
  • POST /api/funcionarios/ - Criar novo usuário
  • GET /api/funcionarios/{cpf} - Obter usuário específico
  • PUT /api/funcionarios/{cpf} - Atualizar usuário
  • DELETE /api/funcionarios/{cpf} - Remover usuário

Autenticação

  • POST /api/login/gerentes/ - Login de gerentes
  • POST /api/login/funcionarios/ - Login de funcionários

Relatórios

  • GET /api/relatorio - Gerar relatório
  • GET /api/erro - Listar erros
