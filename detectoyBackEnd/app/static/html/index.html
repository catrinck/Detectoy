Detectoy API

API para detecção de defeitos em dispositivos móveis e gerenciamento de usuários

Documentação

Acesse a documentação interativa em: /docs ou em /redoc

Webcam

  • GET /api/v1/webcam/start

    Inicia a captura da webcam

  • GET /api/v1/webcam/stop

    Para a captura da webcam

  • GET /api/v1/webcam/frame

    Captura um frame da webcam e faz a detecção

Nota: A webcam captura frames a cada 500ms e salva automaticamente quando detecta defeitos.

Detecção de Imagens

  • POST /api/v1/detections

    Upload de imagem para detecção de defeitos

Gerenciamento de Gerentes

  • GET /api/v1/gerentes/ - Listar todos os gerentes
  • POST /api/v1/gerentes/ - Criar novo gerente
  • GET /api/v1/gerentes/{cpf} - Obter gerente específico
  • PUT /api/v1/gerentes/{cpf} - Atualizar gerente
  • DELETE /api/v1/gerentes/{cpf} - Remover gerente

Gerenciamento de Funcionários

  • GET /api/v1/funcionarios/ - Listar todos os funcionários
  • POST /api/v1/funcionarios/ - Criar novo funcionário
  • GET /api/v1/funcionarios/{cpf} - Obter funcionário específico
  • PUT /api/v1/funcionarios/{cpf} - Atualizar funcionário
  • DELETE /api/v1/funcionarios/{cpf} - Remover funcionário

Autenticação

  • POST /api/v1/login/gerentes/ - Login de gerentes
  • POST /api/v1/login/funcionarios/ - Login de funcionários

Relatórios

  • GET /api/v1/relatorio - Gerar relatório
  • GET /api/v1/erro - Listar erros
