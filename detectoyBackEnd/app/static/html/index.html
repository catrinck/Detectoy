Detectoy API

API para detecção de defeitos em dispositivos móveis e gerenciamento de usuários

Documentação

Acesse a documentação interativa em: /docs

Detecção de Imagens

  • POST /api/v1/detections

Gerenciamento de Gerentes

  • GET /api/gerentes/ - Listar todos os gerentes
  • POST /api/gerentes/ - Criar novo gerente
  • GET /api/gerentes/{cpf} - Obter gerente específico
  • PUT /api/gerentes/{cpf} - Atualizar gerente
  • DELETE /api/gerentes/{cpf} - Remover gerente

Gerenciamento de Funcionários

  • GET /api/funcionarios/ - Listar todos os funcionários
  • POST /api/funcionarios/ - Criar novo funcionário
  • GET /api/funcionarios/{cpf} - Obter funcionário específico
  • PUT /api/funcionarios/{cpf} - Atualizar funcionário
  • DELETE /api/funcionarios/{cpf} - Remover funcionário

Autenticação

  • POST /api/login/gerentes/ - Login de gerentes
  • POST /api/login/funcionarios/ - Login de funcionários

Relatórios

  • GET /api/relatorio - Gerar relatório
  • GET /api/erro - Listar erros
